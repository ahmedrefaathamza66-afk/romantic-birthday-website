# 🎂 موقع عيد ميلاد خطيبتك - المواصفات

## 1. Concept & Vision
موقع رومانسي أنيق لعيد ميلاد الخطيبة، يجمع بين الدفء العاطفي والتصميم العصري. الموقع يحكي قصة حب من أول لقاء حتى اليوم، مع تأثيرات بصرية راقية وتجربة تفاعلية تجعل المناسبة مميزة.

## 2. Design Language

### الألوان
- **Primary:** #C8526B (وردي غامق)
- **Secondary:** #D4AF37 (ذهبي)
- **Background:** #FFF5F7 (وردي فاتح جداً)
- **Text:** #2D2D2D
- **Accent:** #FFB6C1 (وردي فاتح)
- **Dark Rose:** #9B2C44
- **Gold Light:** #F0D080

### الخطوط
- **العربي:** Tajawal (Google Fonts) - للأقسام
- **توقيع:** Dancing Script - للرسائل والتواقيع

### الحركة
- Transitions: 0.3s ease للـ hover
- Animations: 0.6-0.8s للـ reveal
- Particles: حركة بطيئة للأعلى

## 3. Layout & Structure

### الأقسام (8 أقسام)
1. **Hero** - الصفحة الرئيسية مع gradient متحرك
2. **Story/Timeline** - خط زمني للذكريات
3. **Gallery** - شبكة صور Masonry-style
4. **Video** - مشغل فيديو
5. **Love Reasons** - بطاقات أسباب الحب
6. **Letter** - رسالة قديمة التصميم
7. **Counter** - عداد الوقت معاً
8. **Footer** - ختام الموقع

## 4. Features & Interactions

### Custom Cursor
- قلب يتبع الماوس 💗

### Loading Screen
- شاشة تحميل 3 ثواني مع progress bar

### Particles
- قلوب ونجوم طايرة في الخلفية

### Confetti
- انفجار كرات ملونة عند التحميل

### Scroll Animations
- Fade + slide للأقسام

### Lightbox
- عرض الصور مكبرة

### Video Player
- دعم YouTube Embed + فيديو محلي (MP4)
